About Me

Sofyan

Front-End Developer

Saya Sofyan Yunianto, lulusan S1 Teknologi Informasi yang memiliki ketertarikan kuat dalam bidang Frontend Development. Selama lebih dari satu tahun saya berpengalaman sebagai freelancer, mengerjakan berbagai proyek antarmuka web yang fokus pada desain responsif, performa tinggi, dan pengalaman pengguna yang intuitif. Saya terbiasa menggunakan teknologi modern seperti React, Tailwind CSS, dan JavaScript, serta memiliki kemampuan untuk berkolaborasi dengan tim lintas fungsi dalam mewujudkan ide menjadi produk digital yang fungsional dan menarik.

Project

Web OWS Pertamina

Mengembangkan aplikasi web internal untuk mendukung proses analisis beban kerja (Work Load Analysis) berbasis metode Operator Work Sampling (OWS) di lingkungan Pertamina. Berperan sebagai Frontend Developer, saya bertanggung jawab dalam membangun antarmuka pengguna yang responsif, intuitif, dan mudah digunakan, menggunakan React dan Tailwind CSS. Proyek ini membantu tim operasional dalam mengumpulkan, memvisualisasikan, dan menganalisis data aktivitas operator secara efisien melalui dashboard interaktif.
